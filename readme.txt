This plugin helps to manage leads from Contact form 7.

== Description ==
This fabulous plugin has been developed by <a href="https://www.yudiz.com/">Yudiz Solutions Ltd.</a> LA CRM Integration with Contact Form helps to send contact form data to LA CRM.

**How to use this plugin?**
Follow the below <b>steps</b>:
1) Install and activate the plugin.
2) After successful activation, 'CF7 LA CRM API Settings' is added to admin menu in Wordpress dashboard.
3) Here, add User Token and API code from LA CRM Accounts page.
4) 'LA CRM' tab will be added to Contact Form 7 tabs and from there we can allow CRM to pass data from contact form 7 to LA CRM.
5) There are two ways to manage contacts: Create Contact & Create Company.
6) Here, we can pass specific contact form fields to LA CRM fields.
7) There are also custom fields, where we can pass specific contact form fields to custom fields of company/contact in CRM.

The idea behind the development of the plugin:
To easily get & manage (company or contact) leads from Contact form 7.

Hope you find this plugin useful. <a href="https://www.yudiz.com/get-in-touch/">Contact us</a> if you want to develop such custom wordpress plugin.


== Installation ==

1. Upload `la-crm-integration-with-contact-form.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
